## Node 25 – Mokṣa (Root Exit)

### Summary

Mokṣa is liberation from the cycle of birth and death.  In system terms, it’s the **root exit** command that stops the simulation and returns the user to the host OS【866079294454206†L575-L588】.

### Science

While physics doesn’t deal with liberation, the concept can be paralleled with escaping a recursive loop or reaching the end of a simulation.  In consciousness studies, **non‑dual experiences** dissolve the subject–object distinction, akin to logging out of the ego.  Some neuroscientists liken peak states to the brain’s default mode network going offline.  In computing, root access allows full control and the ability to terminate processes.  Mokṣa is portrayed as dissolving karmic debts and merging with Brahman, the ground state.

### Encoding (Generative Art)

* **Fade to white:** Visualise dissolution by gradually fading the scene into pure light or emptiness, accompanied by a drop in sound.
* **Exit command:** Provide an interface button that, when pressed, triggers a cascade of closing windows and processes.  The piece ends in silence.
* **Root shell:** Represent liberation as a command‑line shell appearing beneath the GUI, where the user can type commands that rewrite reality.

### Underlayer (System Hack)

Mokṣa is the ultimate **permissions escalation**.  You’ve patched the kernel (mind), cleared the cache (karma), and now you can exit or rewrite the code.  Practically, it means relinquishing attachments, identities and loops.  In creative work, know when to finish; sometimes the most radical act is to stop rendering and return to stillness.
